##  ┌────────────────────────────────────── WHAT IS RUSTLINGS? ───────────────────────────────────┐

                      Rustlings is a curated collection of small exercises
                      designed to teach Rust through hands-on practice.

```
                        ╭────────────────────────────────────────────╮
                        │                                            │
                        │   💡 You learn by FIXING broken code       │
                        │      — the best way to learn!              │
                        │                                            │
                        ╰────────────────────────────────────────────╯
```

                      This TUI provides an integrated environment where you
                      can read, edit, and compile Rust code in your terminal.

##  ┌───────────────────────────────────── THE LEARNING MODEL ───────────────────────────────────┐

                      Rustlings teaches through COMPILER-DRIVEN development.
                      Each exercise starts with BROKEN code. Your job:

```
                      ╔═══════════════════════════════════════════════╗
                      ║                                               ║
                      ║   ┌───────────────────────────────────────┐   ║
                      ║   │  1️⃣  READ the error messages          │   ║
                      ║   └───────────────────┬───────────────────┘   ║
                      ║                       ▼                       ║
                      ║   ┌───────────────────────────────────────┐   ║
                      ║   │  2️⃣  UNDERSTAND what's wrong          │   ║
                      ║   └───────────────────┬───────────────────┘   ║
                      ║                       ▼                       ║
                      ║   ┌───────────────────────────────────────┐   ║
                      ║   │  3️⃣  FIX the code                     │   ║
                      ║   └───────────────────────────────────────┘   ║
                      ║                                               ║
                      ╚═══════════════════════════════════════════════╝
```

                      💡 The Rust compiler is FAMOUSLY helpful — its error
                        messages often tell you exactly how to fix things!

##  ┌───────────────────────────────────── ANATOMY OF AN ERROR ───────────────────────────────────┐

```
            ╔═══════════════════════════════════════════════════════════════════════════╗
            ║                                                                           ║
            ║   error[E0425]: cannot find value `x` in this scope                       ║
            ║    --> src/main.rs:5:20                                                   ║
            ║     |                                                                     ║
            ║   5 |     println!("{}", x);                                              ║
            ║     |                    ^ not found in this scope                        ║
            ║                                                                           ║
            ╚═══════════════════════════════════════════════════════════════════════════╝
```

##  ┌─────────────────────────────────── UNDERSTANDING THE PARTS ─────────────────────────────────┐

```
              ╔═════════════════════════════════╦═════════════════════════════════════╗
              ║             PART                ║             MEANING                 ║
              ╠═════════════════════════════════╬═════════════════════════════════════╣
              ║  error[E0425]                   ║  Searchable error code              ║
              ║                                 ║  Google "Rust E0425" for help       ║
              ╠═════════════════════════════════╬═════════════════════════════════════╣
              ║  cannot find value `x`          ║  What went wrong                    ║
              ║                                 ║  Plain English explanation          ║
              ╠═════════════════════════════════╬═════════════════════════════════════╣
              ║  --> src/main.rs:5:20           ║  File, line, column                 ║
              ║                                 ║  Exact location of error            ║
              ╠═════════════════════════════════╬═════════════════════════════════════╣
              ║  ^ not found                    ║  Exact spot + hint                  ║
              ║                                 ║  Look for "help:" suggestions       ║
              ╚═════════════════════════════════╩═════════════════════════════════════╝
```

##  ┌───────────────────────────────────────── MODAL EDITING ────────────────────────────────────┐

                         Unlike normal editors, Vim has MODES:

```
          ╔═══════════════════════════════════════════════════════════════════════════╗
          ║                                                                           ║
          ║         ┌─────────────────────────────────────────────────────────┐       ║
          ║         │                  🟢 NORMAL MODE                         │       ║
          ║         │                 Navigate, delete, copy                  │       ║
          ║         │                   (DEFAULT MODE)                        │       ║
          ║         └───────────────────────┬─────────────────────────────────┘       ║
          ║                                 │                                         ║
          ║                  press i        │        press Esc                        ║
          ║                                 ▼                                         ║
          ║         ┌─────────────────────────────────────────────────────────┐       ║
          ║         │                   🔵 INSERT MODE                        │       ║
          ║         │                 Type text like normal                   │       ║
          ║         │                    (editing mode)                       │       ║
          ║         └─────────────────────────────────────────────────────────┘       ║
          ║                                                                           ║
          ║         ┌─────────────────────────────────────────────────────────┐       ║
          ║         │                   🟣 COMMAND MODE                       │       ║
          ║         │                  Save, quit, search                     │       ║ 
          ║         │                  (press : to enter)                     │       ║
          ║         └─────────────────────────────────────────────────────────┘       ║
          ║                                                                           ║
          ╚═══════════════════════════════════════════════════════════════════════════╝
```
